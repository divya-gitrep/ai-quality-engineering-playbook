# What is ChatGPT?

## Describe the product

ChatGPT is an AI chatbot by OpenAI which is used by users to ask questions about any topic and receive a newly generated answer. Release in November 2022, it's built on the GPT  (Generative Pre-trained Transformer) architecture. 

Let's breakdown ChatGPT:

Chat : Chatbot interface used by users to input a question or prompt to interact with AI

Generative: AI generates new content rather than retrieving existing content

Pre-trained: Model was trained on data from : internet, books, users, AI trainers and reserachers

Transformer: Advanced Deep Learnng architecture ChatGPT uses to understand context and     relationships between words

Key features:

1. Writes code, helps debug and resolve issues
2. Explains complex concepts and help learn different topics
3. Assists with analysis, problem-solving and brainstorming
4. Helps plan activities and suggest alternative options

Evolution:

1. ChatGPT can see, hear and speak. It can see uploaded images, it can hear by receiving prompts through voice mode, and have speak by answering in audio format.
2. ChatGPT can browse the internet for real time information and events
3. Custom GPTs: Users can create their own custom GPTs to accomplish specific tasks (for example, a meal planner)

Critical examination: 

Since ChatGPT is trained on data from internet, books and humans, it doesn't have its own judgement about whether the answeres it produces are correct or not. It can produce incorrect information confidently. 

ChatGPT doesn't think or understand lie a human, so it lacks contextual meaning.It may give incorrect answers, and may even hallucinate.

# Business Value
## What problem does it solve?

ChatGPT solves following probelems:

1. Difficulty in gathering knowledge from multiple sources like books, websites and experts
2. Long time to summarize the informarion
3. Confusing and daunting learning journey
4. Struggle with productivity
5. Lack of access to experts like career coach, coding assistant, brainstorming partner 


What happens if the problem is not solved?

1. Difficulty in gathering knowledge from multiple sources like books, websites and experts
 
Before ChatGPT, this problem used to take long time (days,months or even years).
It may also cost money if user has to buy books, access to specific record on websites or pay fees to consult experts.

2. Long time to summarize the informarion

After user has collected massive amount of information, it make take even more time to summarize this information.

3. Confusing and daunting learning journey

A user has to go through searching several web resources or consult with experts to find the right learning resources. Even if the user gets these resources, they may be generic and not fit user's specific profile. User stays frustrated in search of personalised learning plan.

4. Struggle with productivity

User struggles with different time consuming and mundane tasks like writing emails, editing documents, feeling stuck with a task or not knowing where to start a project

5. Lack of access to experts like career coach, coding assistant, brainstorming partner 

User's faces obstacles in personal and professional growth
